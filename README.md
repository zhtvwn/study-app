# 📚 初中生学习助手

一个专为初中生设计的在线学习程序，包含组卷功能和每日做题打卡功能。

## ✨ 功能特点

### 1. 每日练习
- 支持按科目、难度筛选题目
- 自动记录答题情况
- 答完显示答案解析
- 错题自动加入错题本

### 2. 智能组卷
- 自定义试卷名称
- 选择科目和题型分布
- 调整难度比例
- 支持打印试卷
- 可在线答题

### 3. 题库管理
- 内置25道精选题目
- 支持添加自定义题目
- 支持选择题、填空题、解答题
- 可按科目/题型/难度筛选

### 4. 错题本
- 自动记录错题
- 标记复习状态
- 随时查看复习

### 5. 每日打卡
- 记录学习天数
- 显示连续打卡天数
- 激励持续学习

### 6. 学习统计
- 本周/本月/总做题数
- 正确率统计
- 学习记录历史
- 可视化图表

## 🚀 使用方法

### 直接打开
1. 下载项目文件
2. 双击 `index.html` 用浏览器打开
3. 无需安装，无需联网

### 支持的浏览器
- Chrome
- Edge
- Firefox
- Safari

## 💾 数据存储

所有数据保存在浏览器本地存储（LocalStorage）中：
- 题库数据
- 学习记录
- 打卡记录
- 错题本
- 保存的试卷

**注意：** 清除浏览器数据会导致学习记录丢失！

## 🎯 使用建议

1. **每日练习**：建议每天做10-15道题，保持学习节奏
2. **错题复习**：定期查看错题本，巩固薄弱知识点
3. **组卷练习**：周末可以生成一套完整试卷模拟考试
4. **坚持打卡**：养成每日学习习惯，连续打卡更有成就感

## 🔧 自定义题库

你可以通过"题库管理"页面添加自己的题目：
1. 点击"添加题目"
2. 选择科目、题型、难度
3. 填写题目内容
4. 输入正确答案和解析
5. 保存即可使用
